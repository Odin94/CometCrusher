# Comet Crusher

![](.readme_assets/CometCrusher.png)

## How to run
* Install python 2.7
* Install pygame 2.0.3 (or use requirements.txt)
* Run `python CometCrusher.py`

## How to play
* Move right/left with `A`/`D`
* Shoot with `Space`
* Hit comets before they reach the bottom of the screen to collect points
